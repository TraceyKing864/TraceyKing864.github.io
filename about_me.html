Programming

Programming is a hobby I started to get into through games originally. Specifically through Warcraft 3 with its' extensive level editor. It allowed you to edit terrain, data values, models, scripts, events, and was my first real exposure to coding. I enjoyed the creational aspect and freedom given by this game and wanted to learn more. In middle school I took a class in web design using HTML and CSS in Dreamweaver. At the time, the code made sense to me, however I didn't think I would want to make a career out of coding anything. It wasn't until High School when I actually learned a proper programming language that I started to really enjoy it. I started learning C and C++ over the summer of my junior and senior years of high school. It was interesting to me, but I wasn't sure I wanted to go into Computer Science still, as I thought of it more as a hobby. After getting to college and realizing I didn't want to be an Engineer, I switched majors to Computer Science and have throughly enjoyed it since.

My programming interests include game design, procedurally generated content, and combining code with art.
